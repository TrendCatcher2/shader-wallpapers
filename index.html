shader_wallpapers · v1.0 · mit

Five wallpapers that react when you touch them.

A small set of interactive WebGL shaders built for bug-hunter desks. Each one tracks your cursor and detonates on click. Free to download as 4K-ish PNGs, MP4 loops, or the live HTML. No analytics, no dependencies, ~150 lines of GLSL each.

5 wallpapers 2560 × 1440 png live html · webgl mit license

Install

// static · easiest

PNG as desktop background

Download the PNG. macOS: System Settings → Wallpaper → Add Photo. Windows: right-click the file → Set as desktop background.

// animated · windows

Live HTML via Lively

Install Lively Wallpaper (free). Drag any .html file from wallpapers/ into Lively. Mouse + click stay interactive.

// animated · macos

Live HTML via Plash

Install Plash (free). Add Website → point it at the live URL on this page, or a local file:// path.
